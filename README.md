# MedKit｜白衣のポケット計算ノート

腎機能・検査値・輸液の計算を、ブラウザだけで。
**完全オフラインで動く、HTML1ファイルの医療系計算ノート**です。

> 🧑‍⚕️ 作者: **アースキー**（薬剤師）— [作者について](https://earthkey-portfolio.vercel.app/)
> ⚠️ 本ツールは**個人開発・学習用**です。作者の所属施設・団体とは一切関係ありません。計算結果はすべて参考値であり、臨床判断は必ず添付文書・ガイドライン・成書で確認してください。

## 使い方

1. [`index.html`](index.html) をダウンロードしてブラウザで開く（ダブルクリックでOK。ネット接続・インストール・登録すべて不要）
2. 上部タブで「**腎機能**」「**検査値**」「**輸液**」を切り替える
3. 数値を入力して「計算」ボタンを押す（Enterキーでも可）
4. 各計算カードの「**▸ 計算式を見る**」をクリックすると、計算の根拠になっている式が表示されます

## できること

### 🫘 腎機能タブ

| 項目 | 式 |
|---|---|
| eGFRcr（標準化） | 日本人GFR推算式（JSN 2008）: 194 × Cr⁻¹·⁰⁹⁴ × 年齢⁻⁰·²⁸⁷（女性 × 0.739） |
| 個別化eGFR | 標準化eGFR × BSA / 1.73 |
| eCCr（個別化） | Cockcroft-Gault式: (140−年齢) × 体重 / (72 × Cr)（女性 × 0.85） |
| 標準化eCCr | eCCr × 1.73 / BSA |
| eGFRcys（標準化） | 日本人向け式: (104 × CysC⁻¹·⁰¹⁹ × 0.996^年齢 [× 0.929 女性]) − 8 |
| 体表面積 (BSA) | Du Bois式: 0.007184 × 身長^0.725 × 体重^0.425 |
| 理想体重 | BMI 22 基準: 22 × 身長(m)² |

標準化eGFRにはCKD重症度分類のGFR区分バッジ（G1〜G5）を表示します。
参考: 日本腎臓病薬物療法学会「eGFR・eCCrの計算」 https://jsnp.org/egfr/

### 🧪 検査値タブ

| 計算 | 式 |
|---|---|
| 補正Ca（Payne式） | 実測Ca + (4 − Alb) |
| mEq ⇔ mg 換算 | mEq = mg ÷ 原子量 × 価数（Na/K/Ca/Mg/Cl のプリセット） |
| Na補正速度チェック | \|目標Na − 現在Na\| ÷ 予定時間。24h換算し一般的な目安（10〜12mEq/L/24h）と照合 |
| 水分欠乏量（高Na血症時） | 体重 × 体内水分比 × (現在Na ÷ 140 − 1) |
| 血漿浸透圧（概算） | 2×Na + 血糖/18 + BUN/2.8（正常域目安 275〜295 mOsm/kg） |
| TSAT | 血清鉄 ÷ TIBC × 100（%）。フェリチン併用で鉄欠乏の目安判定 |

### 💧 輸液タブ

| 計算 | 式 |
|---|---|
| 点滴速度 | 速度(mL/h) = 総量 ÷ 時間／滴数(滴/分) = 速度 × 滴数係数 ÷ 60（成人20滴・小児60滴） |
| 希釈・濃度計算 | 濃度(mg/mL) = 薬剤量 ÷ 総量。3項目中2つを入力すると残り1つを算出 |
| 投与可能時間 | 残量 ÷ 流速 |

## プライバシーとセキュリティ

- **外部通信ゼロ** — fetch / XMLHttpRequest / CDN / 外部フォント・画像を一切使いません
- **入力データを保存しません** — localStorage / sessionStorage / Cookie 不使用。ページを閉じればすべて消えます
- **CSPメタタグ**で外部通信・フォーム送信・外部読み込みをブラウザレベルでも遮断しています
- 氏名・IDなど患者個人情報の入力欄はありません（入力しないでください）
